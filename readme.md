## Mongo file storage

Store your files in mongodb directly

### Steps

1. Install dependencies

```bash
npm i
```

2. Setup env

```bash
PORT=3000
PROJECT_NAME=MY Project
DB=mongodb://localhost:27017/db-storage
```

3. Start server

```bash
node app.js
```
